Welcome to Dolstagis.Web

Congratulations! Your new Dolstagis.Web project is ready to go.

To get started, create a feature and one or more controllers: 

using System;
using Dolstagis.Web;

public class MyFeature: Feature
{
    public MyFeature()
    {
        AddController<MyHomePage>();
    }
}


[Route("/")]
public class MyHomePage: Controller
{
    public object Get()
    {
        return Content("Hello world!");
    }
}

For more information, see the project wiki.

This message will disappear when you define a controller for your home page.
