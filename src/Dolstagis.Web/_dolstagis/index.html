Welcome to Dolstagis.Web

Congratulations! Your new Dolstagis.Web project is ready to go.

To get started, create a module and one or more handlers: 

using System;
using Dolstagis.Web;

public class MyModule: Module
{
    public MyModule()
    {
        AddHandler<MyHomePage>();
    }
}


[Route("/")]
public class MyHomePage: Handler
{
    public object Get()
    {
        return Content("Hello world!");
    }
}

For more information, see the project wiki.

This message will disappear when you define a handler for your home page.
